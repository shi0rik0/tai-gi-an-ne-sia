# 臺灣閩南語按呢寫：第 272 期

## 【佮意】kah-ì

### 對應華語

中意

### 用例

有佮意、無佮意

### 異用字

愜意、合意

### 民眾建議

合意

### 用字解析

Kah的漢字寫成「佮」，因此用「佮意」來寫「kah-ì」在發音上就完全合理。如果用民眾建議的「合意」，雖然在意思上比較容易理解，但「合」字在臺灣閩南語中本來就有兩個常用的發音，一個是文讀音的「ha̍p」，常用的例子有「合作」、「聯合」；另一個是白讀音的「ha̍h」，常用的例子有「合軀」（合身）、「有合」（相配）。如果把「kah-ì」寫成「合意」，一方面造成「合」字發音太多（三個）的負擔，同時還可能造成某些用法發音上的困擾。例如：「這領衫無合我的意」，其中的「合」正確的發音是「ha̍h」。但如果「kah-ì」也寫成「合意」，那麼上面的例子就會讓人很難確定「合」字到底要說成「kah」或「ha̍h」。基於以上的理由，標記「kah-ì」時，「佮意」優於「合意」，因此把「佮意」定為推薦用字。

## 【家己】ka-tī/ka-kī

### 對應華語

自己

### 用例

家己人、靠家己

### 異用字

家治

### 民眾建議

己自、家自

### 用字解析

華語所說的「自己」，在臺灣閩南語中，有人說成「ka-tī」，也有人說成「ka-kī」，漢字則多寫作「家己」（蔡培火、王育德、胡鑫麟、陳修、楊青矗、董忠司等人所編的《臺灣閩南語辭典》都寫作「家己」）。「家」臺灣閩南語發音有「ka」（文讀音）和「ke」（白讀音）兩種；「己」的臺灣閩南語發音為「kí」。所以用「家己」來寫「ka-tī」」或「ka-kī」，發音上非常接近，語義上也容易理解，所以各家辭典都這麼寫。若寫成民眾建議的「己自」或「家自」，發音分別是「kí-tsū」和「ka-tsū」，不可能說成「ka-tī」或「ka-kī」，文獻上也沒有這樣的紀錄，因此尊重臺灣閩南語文獻的傳統，還是寫成「家己」最為理想。
