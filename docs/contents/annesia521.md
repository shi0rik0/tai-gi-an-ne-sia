# 臺灣閩南語按呢寫：第 521 期

## 【褫】thí

### 對應華語

張開、展開

### 用例

目睭褫袂開

### 用字解析

華語的「張開、展開」，臺灣閩南語說成thí，漢字寫成「褫」。用法如：「電火光甲目睭褫袂開。」Tiān-hué kng kah ba̍k-tsiu thí bē khui.（電燈亮得眼睛睜不開。）「咧落雨矣，雨傘緊褫開。」Teh lo̍h-hōo--ah, hōo-suànn kín thí--khui.（下起雨來了，傘趕緊撐開。）

以「褫」做為thí的用字是有道理的。依據《廣韻》，「褫」的音讀是「敕爾切」，意思是「奪衣也」。「褫」在臺灣閩南語裡的音讀就是thí。「褫」的本義是「搶奪衣服」，所以有「剝奪」的引申義（如「褫奪公權」thí-tua̍t kong-khuân），「搶奪、剝奪」必須讓物和所有人分開，所以又引申為「張開、展開」，而這正是臺灣閩南語「褫」thí的用法（如「褫雨傘」thí hōo-suànn（開傘））。「褫」thí和「開」khui常結合成「褫開」thí-khui/thí--khui在口語中使用。

文獻中如《廈門音新字典》、《彙音妙悟》都以「褫」為thí的用字，因為是民間習用的字，因此獲選為推薦用字。
