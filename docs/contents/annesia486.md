# 臺灣閩南語按呢寫：第 486 期

## 【怨嘆】uàn-thàn

### 對應華語

怨恨悲嘆

### 用例

怨嘆月暝、毋免怨嘆

### 用字解析

華語的「怨恨悲嘆」臺灣閩南語說成uàn-thàn，漢字寫成「怨嘆」，用法如：「這是你的命，毋免怨嘆。」Tse sī lí ê miā, m̄-bián uàn-thàn.（這是你的命，用不著怨恨悲嘆。）「騙都去予騙去矣，怨嘆敢有路用？」Phiàn to khì hōo phiàn--khì--ah, uàn-thàn kám ū-lōo-īng?（騙都被他騙了，怨恨悲嘆有用嗎？）

依據《廣韻》，「怨」的音讀是「於願切」，意思是「恨也，說文：『恚也』」；「嘆」的音讀是「他旦切」，意思是「歎息」。所以「怨嘆」就是uàn-thàn的本字，因為心中有恨時，不免會抱怨嘆息，所以就把「怨」和「嘆」合在一起，構成「怨嘆」這樣的並列複合詞。「怨嘆」不僅是本字，也是文獻中使用已久的用字，所以獲選為推薦用字。
