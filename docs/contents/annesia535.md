# 臺灣閩南語按呢寫：第 535 期

## 【坉】 thūn

### 對應華語

填入

### 用例

坉井、坉塗、坉本

### 用字解析

華語的「填入」，臺灣閩南語說成thūn，漢字寫成「坉」。「坉塗」thūn thôo是填入泥土，用法如：「草埔有一窟，愛坉塗才會平。」Tsháu-poo ū tsi̍t khut, ài thūn thôo tsiah ē pênn.（草地上有一個窟窿，要填土才會平。）「坉井」thūn-tsénn是把土石填入井中，將井填平，例如：「鼓井無咧用矣，驚囡仔跋落去，逐家決定欲坉井。」Kóo-tsénn bô teh iōng--ah, kiann gín-á pua̍h--lo̍h-khì, ta̍k-ke kuat-tīng beh thūn-tsénn.（井已經不用了，怕小孩跌下去，大家決定要填井。）「坉本」thūn-pún則是引申的抽象用法，指把資金投入有風險的事業中，像是：「伊投資彼間公司，到這馬猶一直咧坉本。」I tâu-tsu hit king kong-si, kàu tsit-má iáu it-ti̍t teh thūn-pún.（他投資那家公司，到現在還一直投入資金。）

「坉」在《廣韻》中的音讀是「徒渾切」，意思是「以草裹土築城及填水也。」在《集韻》中，「坉」除了「徒渾切」，還有「杜本切」的音讀，意思則是「艸土填水曰坉。」從音、義上來說，以「坉」做為thūn的用字都說得通。雖然「坉」不是常用字，但筆畫簡單，書寫容易，再加上「坉」的義符「土」（thūn通常要用「土」）和聲符「屯」（臺灣閩南語發音為tūn, tûn, thûn），都很容易和臺灣閩南語的thūn產生連結，非常有利於學習使用，因此獲選為推薦用字。
