# 臺灣閩南語按呢寫：第 491 期

## 【晏】uànn

### 對應華語

晚、遲

### 用例

晏睏、晏起來

### 用字解析

華語的「晚、遲」臺灣閩南語說成uànn，漢字寫成「晏」，用法如：「你昨暝哪會遐晏睏？」Lí tsa-mê ná ē hiah uànn khùn?（你昨天晚上怎麼那麼晚睡？）「讀冊囡仔毋通遐爾晏起來。」Tha̍k-tsheh gín-á m̄-thang hiah-nī uànn khí--lâi.（求學中的小孩不應該那麼晚起床。）

依據《廣韻》，「晏」的音讀是「烏旰切」，意思是「晚也」。「晏」的文讀音是àn，白讀音是uànn，合乎文白對應規律，例如「安」的文讀音是an（「安全」an-tsuân），白讀音是uann（「同安」Tâng-uann）。音義相符，可證「晏」就是uànn的本字。

以「晏」為「晚、遲」，在文獻中很常見，例如：《禮記‧內則》：「孺子蚤寢晏起，唯所欲。」又《呂氏春秋˙似順論˙慎小》：「二子待君，日晏，公不來至。」

因為「晏」是uànn的本字，也是文獻中習用的字，所以獲選為推薦用字。
