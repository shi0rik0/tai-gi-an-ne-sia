# 臺灣閩南語按呢寫：第 496 期

## 【身軀】sin-khu

### 對應華語

身體

### 用例

洗身軀、身軀勇壯

### 用字解析

華語的「身體」，臺灣閩南語說成sin-khu，漢字寫成「身軀」。用法如：「運動了後規身軀汗，愛緊去洗身軀。」Ūn-tōng liáu-āu kui-sin-khu kuānn, ài kín khì sé sin-khu.（運動之後全身大汗，要趕快去洗澡。）

「身軀」是漢語自古以來就有的詞，《文選》王褒‧〈洞簫賦〉：「託身軀於后土兮，經萬古而不遷。」《初刻拍案驚奇》卷四：「猙獰胡貌，劣撅身軀。」早期白話文學《水滸傳》第三八回中：「李逵撲翻身軀便拜。」都有「身軀」一詞。現代華語口語中較少用到「身軀」，但書面語中還是很常見，例如：「拖著疲憊的身軀。」因此「身軀」就是臺灣閩南語sin-khu的本字。

由於「身軀」是口語中常用的詞，又因為「身」的韻尾是舌尖鼻音-n，而「軀」的聲母是舌根塞音kh-，為了方便發音，口語中「身軀」中「身」的韻尾常被「軀」的聲母同化，所以實際發音都發成sing-khu。此外「洗身軀」（洗澡）也是日常口語常用的詞，因為「洗」和「身」的聲母都是舌尖擦音s-，說話時，為了求快速、方便，多數人會把「洗身軀」說成sé-hing-khu。
